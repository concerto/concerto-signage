Server Requirements

Installing Concerto requires some basic server administration experience. You'll need to have installed and be familiar with the setup of:

  • A *nix OS
  • Ruby 1.9.x or better
  • A web server (such as Apache, nginx, etc)
  • A connector for serving Ruby on Rails applications such as Passenger (if the webserver doesn't support Rails natively)
  • A relational database (MySQL, Postgresql, or even SQLite will do)
